## Sezione 1: Introduzione

Qui puoi scrivere qualsiasi contenuto in **Markdown**. Puoi usare:

- Liste puntate
- **Testo in grassetto**
- *Testo in corsivo*
- `Codice inline`

## Sezione 2: Esempi di codice

Puoi anche aggiungere blocchi di codice:

```javascript
function saluta(nome) {
    console.log(`Ciao ${nome}!`);
}

saluta("Mondo");
```

```python
def saluta(nome):
    print(f"Ciao {nome}!")

saluta("Mondo")
```
